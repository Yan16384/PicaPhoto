# PicaPhoto
基于 Python + Tkinter 开发的可视化媒体文件管理工具，支持图片/视频分类整理、相册管理、双主题切换、批量操作与快捷键。

## 功能亮点
- 图片/视频分类、筛选、预览
- 相册管理、批量移动、撤销操作
- 亮色 / 暗黑双主题
- 快捷键操作、自动刷新
- 跨平台支持（Windows / macOS / Linux）

## 执行环境（必要依赖）
### 1. Python 版本
Python 3.7 及以上

### 2. 必装第三方库
```bash
pip install pillow

如需视频预览，安装：pip install opencv-python

系统依赖（Linux / Ubuntu）：sudo apt-get install python3-tk

操作逻辑：
点击「选择目录」指定媒体文件夹，程序自动扫描所有图片与视频
单图模式：滚轮 / 方向键快速切换文件，支持旋转、镜像预览
按 Tab 进入预览模式：查看缩略图、多选文件、批量移动到相册
底部栏新建相册，用数字快捷键快速归类文件
支持 Ctrl+Z 撤销上一步移动操作
左侧目录树可按「全部 / 照片 / 视频」筛选内容
关闭程序自动保存进度、相册、主题配置

快捷键大全
← / →：单图模式切换上一张 / 下一张
Tab：切换单图模式 / 预览网格模式
Ctrl+A：预览模式全选文件
Ctrl+Z：撤销上一次文件移动操作
Esc：退出预览模式 / 关闭弹窗
r / R：单图模式旋转图片
f / F：单图模式镜像翻转图片
数字 1–9：快速移动当前 / 选中文件到对应序号相册
